"Star Search" is another full-stack project that uses React and MongoDB. It acts as an online resume for aspiring or established thespians.

It uses Express for server routing and leverages token authentication via JWTs to ensure that users only have access to their own data. It also consumes a RESTful API to pull in movie credit information.
