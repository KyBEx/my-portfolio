"Star Search" is another full-stack project that uses React and MongoDB. It acts as an online resume for aspiring or established thespians.

A user may sign up for the site and create a profile that showcases their bio and movie credits. The movie credits are pulled from TMDB, a RESTful API.

Once a user has signed up, their information is stored in a custom database that is accessible from the search feature on the main page. This information can be accessed by anyone--a fan, a casting director, anyone who accesses the main page. Currently, only one record, Bruce Willis, is in this custom database.

Star Search uses Express for server routing and leverages token authentication via JWTs to ensure that users only have access to their own data. This project was also pair coded with a classmate of mine at V School.
