This project was part of my curriculum at V School. The challenge was to recreate the visual style of the home page for CSS Zen Garden

Originally, I utilized Bootstrap to handle the re-arranging of the layout as the screen size changed. However, recently I revamped the project and used CSS Grid. Grid is a great alternative to Bootstrap and uses pure CSS to manage the visual shuffling of the page contents.
